🔧 Configuración Chrome para Cámara Local

⚠️ Problema: Chrome bloquea el acceso a la cámara en conexiones HTTP locales por seguridad.

📱 Solución para Móvil:

1. Abrir Chrome en tu móvil

Ve a la barra de direcciones y escribe:

chrome://flags/#unsafely-treat-insecure-origin-as-secure

2. Habilitar la opción

Busca la opción y cámbiala a "Enabled"

3. Agregar tu IP

En el campo de texto, agrega:

http://192.168.1.69:8000

4. Reiniciar Chrome

Reinicia completamente la aplicación Chrome

✅ Después de esto: Chrome permitirá el acceso a la cámara en tu servidor local.

🔄 URLs para Probar:

http://192.168.1.69:8000/test-camera.html
http://192.168.1.69:8000/dni-scanner/c29aa162f8175fe5ace4f434bd2fe0d0
💡 Alternativa: Si no puedes acceder a chrome://flags, usa Firefox o Safari que son menos restrictivos con HTTP local.
